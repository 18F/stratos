Use the Cloud Foundry CLI tool to execute commands directly against a Helion Cloud Foundry instance

Before starting follow these steps

1. Download and install the tool via these instructions

2. Log in to your Cloud Foundry$ cf login -a api.[my-cloudfoundry] API endpoint: https://api.[my-cloudfoundry] Email> [my-email] Password> [my-password] Authenticating... OK

3. Change/set the targeted org and/or space $ cf target -o [organization name] -s [space name] API endpoint: https://api.[my-cloudfoundry] (API version: 2.56.0) User: [logged user] Org: [organization name] Space: [space name]

Cluster Commands

REST API
To make a request to the API via curl
cf curl [curl parameters]
Environment variables
CF_COLOR=false Do not colorize output CF_HOME=path/to/dir/ Override path to default config directory CF_PLUGIN_HOME=path/to/dir/ Override path to default plugin config directory CF_STAGING_TIMEOUT=15 Max wait time for buildpack staging, in minutes CF_STARTUP_TIMEOUT=5 Max wait time for app instance startup, in minutes CF_TRACE=true Print API request diagnostics to stdout CF_TRACE=path/to/trace.log Append API request diagnostics to a log file https_proxy=proxy.example.com:8080 Enable HTTP proxying for API requests

Organization Commands

List
To list all orgs
cf orgs
Rename
To rename an org
cf rename-org [org] [name]
Information
To fetch org information
cf org [org]
Users
List each org role's users
cf org-users [org]

Space Commands

List
To list the spaces in the targeted org
cf spaces
Rename
To rename a space
cf rename-space [space] [name]
Information
To fetch space information
cf space [space]
Users
List each space role's users
cf space-users [space]
